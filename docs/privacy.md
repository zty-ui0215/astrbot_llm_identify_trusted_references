# Privacy Policy for Candidate Data

Candidate packages are designed to contain sanitized aggregate evidence only.

Excluded data:

- Raw prompts and completions
- API keys and authorization headers
- Cookies and account identifiers
- Full request URLs with private query strings
- IP addresses and user chat content
- Provider dashboard screenshots or billing data

Maintainers should reject submissions containing sensitive data and ask the reporter to regenerate the package through the plugin exporter.
