# LLM Identify Trusted References

Volunteer-submitted, sanitized official-endpoint fingerprint candidates for `astrbot_plugin_llm_identify`.

Repository target: `https://github.com/zty-ui0215/llm-identify-trusted-references`.

## What This Repository Accepts

This repository accepts candidate fingerprint packages generated only from official model provider APIs, such as:

- OpenAI: `https://api.openai.com/v1`
- Alibaba Cloud Bailian / DashScope OpenAI-compatible API: `https://dashscope.aliyuncs.com/compatible-mode/v1`
- Anthropic, Google Gemini, Mistral, Cohere, DeepSeek, Volcengine Ark, Zhipu, Moonshot, MiniMax, SiliconFlow official API hosts

Candidates are not trusted automatically. Maintainers review them before moving data from `data/candidates/` to `data/accepted/`.

## Submission Paths

1. Open a GitHub Issue with the sanitized JSON package generated by the plugin.
2. Open a pull request adding one package under `data/candidates/<provider>/<model>/<timestamp-or-hash>.json`.

Do not submit raw prompts, completions, API keys, headers, account identifiers, IP addresses, private URLs, or provider dashboard screenshots containing secrets.

## Validation

Run locally:

```bash
python scripts/validate_candidate.py data/candidates
```

GitHub Actions runs the same validator for pull requests.
